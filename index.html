Programación Orientada a Objetos en Python

Una guía completa con ejemplos prácticos

Clases en Python

Las clases son plantillas para crear objetos. Definen atributos y métodos que los objetos tendrán.

# Definición de una clase básica class Persona: def __init__(self, nombre, edad): self.nombre = nombre self.edad = edad def saludar(self): return f"Hola, soy {self.nombre} y tengo {self.edad} años" # Creando una instancia de la clase persona1 = Persona("Ana", 25) print(persona1.saludar())
Salida: Hola, soy Ana y tengo 25 años
Nota: La clase Persona define un template básico con atributos (nombre, edad) y un método (saludar).

Objetos en Python

Los objetos son instancias de clases. Cada objeto tiene sus propios valores de atributos.

# Creando múltiples objetos de la misma clase class Auto: def __init__(self, marca, modelo): self.marca = marca self.modelo = modelo def describir(self): return f"Auto {self.marca} modelo {self.modelo}" # Creando diferentes instancias auto1 = Auto("Toyota", "Corolla") auto2 = Auto("Honda", "Civic") print(auto1.describir()) print(auto2.describir())
Auto Toyota modelo Corolla Auto Honda modelo Civic

Herencia en Python

La herencia permite que una clase herede atributos y métodos de otra clase.

class Animal: def __init__(self, nombre): self.nombre = nombre def hacer_sonido(self): pass class Perro(Animal): def hacer_sonido(self): return f"{self.nombre} dice: ¡Guau!" class Gato(Animal): def hacer_sonido(self): return f"{self.nombre} dice: ¡Miau!" perro = Perro("Bobby") gato = Gato("Michi") print(perro.hacer_sonido()) print(gato.hacer_sonido())
Bobby dice: ¡Guau! Michi dice: ¡Miau!
